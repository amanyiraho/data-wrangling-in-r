What is data wrangling?

A variety of processes designed to transform raw data into more readily used formats

How do you wrangle data?

First question: What exactly, do you want to do?

For example

  • Do you want to filter data based on condition?

  • Do you want to select specific columns out of a bigger a dataframe?

  • Do you want to carry out a computation e.g mean , across groups of a specific variable?

  • Do you want to arrange you dataframe based on a specific column?

Guess what dplyr It provides simple “verbs”, functions each basic verb of data manipulation like the ones highlighted above

What is dplyr?

Dplyr is an R package which implements a grammar of data manipulation, providing a consistent set of verbs that help you solve the most common data manipulation challenges

Most popular functions in the dplyr?

  • mutate() adds new variables that are functions of existing variables.

  • select() picks variables based on their names.

  • filter() picks cases based on their values.

  • summarise() reduces multiple values down to a single summary.

  • arrange() changes the ordering of the rows.

  • group_by() you can perform computation across groups of a variable

Installation

Install dplyr from CRAN

install.packages("dplyr")

Development version from Github

install.packages("devtools")
devtools::install_github("tidyverse/dplyr")

The pipe

Photo by Henry & Co. from unsplash.com

Use %>% or |> as a pipe

dplyr provides the %>% operator from magrittr package.

So the result from one step is then piped into the next step

x %>% f(y) turns into f(x, y)

OR

x |> f(y) turns into f(x, y)

You read the pipe as Then

What about data wrangling with columns:

With columns your vibes may be

  • select() changes whether or not a column is included.
  • rename() changes the name of columns.
  • mutate() changes the values of columns and creates new columns.
  • relocate() changes column order
